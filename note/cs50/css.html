tag类型

css html

#rob h1:hover ?

css选择器

id只能出现一次, 但class可以出现多次

#rob h1 能在 id 直接或间接下级作用
#rob > h1 只能在直接下级作用
#rob ~ h1 在 id 完了才作用
#rob + h1 在id完了紧接第一个才作用
